# Automotive Radiator Heat Exchanger Design and CFD Analysis (Ongoing)

## Overview
This project focuses on the design and development of an automotive radiator heat exchanger using SOLIDWORKS. The objective is to model a radiator assembly and evaluate its thermal performance through CFD-based Conjugate Heat Transfer (CHT) analysis. The current phase involves CAD modeling and assembly development, while simulation studies are planned for future stages.

## Project Objectives
- Design a detailed automotive radiator heat exchanger.
- Prepare geometry for CFD simulation.
- Optimize radiator performance through thermal-fluid analysis.

## Current Progress
- Completed 3D CAD modeling of radiator components.
- Developing radiator assembly in SOLIDWORKS.
- Prepared model for future CFD studies.

## Planned Analysis
The following analyses will be performed in future stages:

- Conjugate Heat Transfer (CHT) Analysis
- Temperature Distribution Analysis
- Coolant Flow Analysis
- Pressure Drop Evaluation
- Heat Transfer Performance Assessment
- Design Optimization
